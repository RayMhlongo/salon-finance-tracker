# Salon Finance Tracker

Static finance tracker for small salons. It runs from `index.html`, stores data in `localStorage`, exports/imports JSON backups, and creates invoice PDFs with jsPDF.

## Deploy to GitHub Pages

1. Upload these files to the `salon-finance-tracker` repository.
2. In GitHub, open **Settings > Pages**.
3. Set source to the `main` branch and root folder.

No backend or paid service is required.
